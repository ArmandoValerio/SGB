## Investiagacion Diferencia entre página web y aplicación web

### Página web
Una página web es un sitio cuyo objetivo principal es **mostrar información** al usuario. Generalmente su contenido es estático o con interacción limitada y no realiza procesos complejos.

**Características:**
- Presenta información fija o informativa  
- Interacción mínima con el usuario  
- No requiere autenticación  
- No maneja lógica de negocio compleja  
- Puede o no usar base de datos  

**Ejemplos:**
- Páginas institucionales  
- Blogs  
- Sitios de presentación de servicios  
![Arquitectura Web](https://www.dominios.mx/wp-content/uploads/2021/01/mx-tipos-sitios.jpg)
---

### Aplicación web
Una aplicación web es un sistema interactivo que permite a los usuarios **realizar acciones**, procesar información y ejecutar operaciones mediante **lógica de negocio**.

**Características:**
- Manejo de usuarios  
- Interacción constante  
- Uso obligatorio de bases de datos  
- Ejecución de procesos como registros, búsquedas y modificaciones  
- Permite escalabilidad  

**Ejemplos:**
- Sistemas administrativos  
- Plataformas educativas  
- Gestores de inventarios  
![Arquitectura Web](https://www.b2chat.io/wp-content/uploads/2022/07/mockup-de-una-aplicacion-web.webp)

## ¿Qué tipo de problemas se resuelven con software?

El software permite resolver problemas relacionados con:
- Desorganización de información  
- Procesos manuales ineficientes  
- Errores humanos en registros  
- Dificultad para acceder a datos  
- Falta de control y seguimiento  

Mediante software se pueden:
- Automatizar procesos  
- Centralizar información  
- Reducir errores  
- Mejorar la eficiencia operativa  
- Facilitar la toma de decisiones

## Arquitectura general de aplicaciones web

### Frontend
Es la parte visual con la que interactúa el usuario.  
Utiliza tecnologías como:
- HTML  
- CSS  
- JavaScript  

Su función es mostrar información y capturar acciones del usuario.

---

### Backend
Es la parte lógica del sistema.  
Se encarga de:
- Procesar solicitudes  
- Aplicar reglas de negocio  
- Conectarse a la base de datos  

Tecnologías comunes:
- PHP  
- Node.js  
- Python  

---

### Base de datos
Es donde se almacena toda la información del sistema, permitiendo:
- Consultas  
- Inserciones  
- Actualizaciones  
- MySQL  

---

### Entornos
- **Desarrollo:** donde se programa y prueba  
- **Producción:** donde el sistema funciona para los usuarios finales
